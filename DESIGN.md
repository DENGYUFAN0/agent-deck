# DESIGN.md — agent-deck 视觉规范（瘦身版）

动 UI 前先读本文件。身份定位：**学术浅色**——干净、克制、投影友好，让内容而非装饰说话。与 compass 家族浅色正典同源。

## 色板（CSS 变量，写死在 :root）

| 变量 | 值 | 用途 |
|------|-----|------|
| `--bg` | `#ffffff` | 幻灯片底 |
| `--ink` | `#1a2332` | 正文 |
| `--muted` | `#5a6b7f` | 次要文字、注释、页脚 |
| `--accent` | `#0f4c81` | 主色：标题下划线、强调、图表主体 |
| `--accent-2` | `#1a7f8e` | 辅色：kicker、进度条、编辑高亮 |
| `--line` | `#dde4ec` | 分隔线、表格边框、图表网格 |
| `--soft` | `#f4f7fa` | 浅底色块、表头 |

语义色（不进变量）：完成 `#e3f3e8/#1d7a43`，进行中 `#fff3d6/#8a6100`，风险 `#fde5e0/#b3372a`。

## 字体与字号（1280×720 设计稿）

- 字体栈：`-apple-system, "Segoe UI", "Microsoft YaHei", "PingFang SC", "Noto Sans CJK SC", "Malgun Gothic", sans-serif`——**禁网络字体**（零依赖红线）。
- 封面 h1 60px / 内页 h2 36px / h3 22px / 正文 24px / 注释 16px / 页脚 13px。
- 正文**不小于 22px**，投影可读是底线。

## 版式规则

- 页面统一内边距 `56px 84px 96px`（底部留给页脚）；封面例外。
- 章节页统一开头：kicker（15px、字距 .22em、辅色、格式 `01 · 章节名`）→ h2 → 主色 64×4px 短下划线。
- 一页一个重点；两栏用 `.cols/.col`（gap 44px）；重点结论用 `.box`（浅底+主色左边线）。
- 页脚 `.slide-foot`：左侧题目缩写，右侧页码（JS 填充）。

## 图表

- 一律内联 SVG，网格线 `--line`，非重点柱 `#b9c9d9`，重点柱 `--accent`；数值标在柱顶，坐标轴文字 14–15px `--muted`。
- 禁位图截图（不可编辑、文件暴涨）。

## Do / Don't

- Do：留白、对齐、同层级样式完全一致。
- Don't：渐变堆叠、阴影堆叠、动画切换、超过两个强调色、emoji 当图标（打印和跨平台渲染不稳）。
- 放映 UI（工具栏/浮层）是深色玻璃风 `rgba(18,24,32,.85)`，与幻灯片浅色内容形成"台上/台下"区隔；打印时必须全部隐藏。
